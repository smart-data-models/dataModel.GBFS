# system_regions
Version: 0.0.1

## Description 

Describes regions for a system that is broken up by geographic or political region. According to the Standard GBFS 2.2

Mapping of the Standard [GBFS 2.2](https://github.com/NABSA/gbfs/blob/v2.2/gbfs.md)
